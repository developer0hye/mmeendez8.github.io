Bio

I am a Machine Learning Research Engineer at Gradiant. I love creating systems based on data that can interact with the real word. My research interests lie in the general area of machine learning, particularly in deep learning and its applications in object detection, image generation and instance segmentation.

Prior to coming to Gradiant, I worked at Desigual Headquarters in Barcelona as a Data Science consultant. I spent one year in Barcelona and one year in the United States (Purdue University) to do a M.Sc. in Artificial Intelligence. I got my B.Sc. from University of A Coruña after an exchange year in Norway.

Other projects

Feature construction and classification on Time Series

Master Thesis - Prof. Bruno Ribero - Purdue University

Collaborated with Federal Aviation Administration (FAA) in PEGASAS project with the aim of enhance general aviation safety. Developed a model which could classify flights in different categories using data obtained from airborne sensors through an unsupervised learning stage and an active learning procedure. Tools: Python, Scikit-Learn, Pandas, C, OpenMPI, Java
Presented in PEGASAS 2017 meeting at Texas A&M University

Instrument recognition using Neural Networks

System for musical instruments recognition using NNs with over 10000 audio samples using Fourier Transform and its harmonics to create the input features. Created our own NN library in Python from scratch.

Corruption simulation in Social Netowrks (Twitter)

Analysis using Twitter publics API where we focus in the relations between politicians in the social network. We performed a Monte Carlo simulation of corruption behavior spreading with the aim of observing how corruption flows starting from some seed users that were imprisoned or involved in some case.
